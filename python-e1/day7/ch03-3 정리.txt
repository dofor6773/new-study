함수란 무엇인가? 
함수를 설명하기 전에 믹서를 생각해 보자. 우리는 믹서에 과일을 넣는다.   
그리고 믹서를 사용해서 과일을 갈아 과일 주스를 만든다.  
우리가 믹서에 넣는 과일은 '입력', 과일주스는 '출력(결괏값)'이 된다

과일(입력)(데이터) ->  믹서기(함수)  -> 주스(출력)(다른 형태)

a함수/f(x) = y, f=함수, x=1일 때 y=2, x=2일 때 y=4, x=3일 때 y=6 -> 2의 배수 함수   
2y = y
b함수/f(x) = y, f=함수, x=1일 때 y=3, x=2일 때 y=6, x=3일 때 6=9 -> 3의 배수 함수   
3y = y
